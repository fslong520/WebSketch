# WebSketch 绘影 - 认证说明

WebSketch 是一款在任意网页上绘图、标注、截图的 Chrome MV3 扩展，v1.5.0。

## 测试步骤

1. 安装扩展后，打开任意网页（如 example.com），点击工具栏 WebSketch 图标，在弹出的面板点击"开启绘图"。

2. 使用工具栏按钮或快捷键切换工具测试：B（画笔）、E（橡皮擦）、R（矩形）、C（圆形）、L（直线）、A（箭头）、T（文字）、H（选择/拖拽）、N（计数圆圈）、V（框选）、X（十字坐标系）。

3. 在网页上绘制图形。选中对象后拖拽移动，按住 Shift 键拖拽可旋转。

4. 按 S 键进入截图模式，拖拽选择区域，松开鼠标自动保存含绘图内容的 PNG。

5. 使用撤销/重做（Ctrl+Z/Y）、取色器、填充/虚线模式、网格辅助线（G）。

## 注意事项

- 文字工具：点击画布定位光标后输入文字，按 Enter 提交。
- 旋转：选中对象后，按住 Shift + 水平拖拽鼠标。
- 压感支持：需要触控笔设备，线条粗细随压力变化。
- 无需登录或特殊账号，打开任意网页即可测试。
